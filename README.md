# Cyan Light Theme for Zed

A port of the [Cyan Light Theme](https://plugins.jetbrains.com/plugin/12102-cyan-light-theme) from the JetBrains Marketplace to [Zed](https://zed.dev).

## Install

Search for "Cyan Light Theme" in Zed's extension marketplace (`zed: extensions`).

## Colors

| Element | Color |
|---|---|
| Editor background | `#f2f3f7` |
| Panel/chrome | `#e4e6eb` |
| Active tab | `#f2f4f5` |
| Borders | `#bec5cd` |
| Text | `#262626` |
| Keywords | `#000080` |
| Strings | `#00733B` |
| Numbers | `#0073E6` |
| Comments | `#808080` |
| Types | `#660E7A` |
| Accent | `#00a9bf` |
